How to install the program:
1. Download all files from my GitHub folder to your local machine as a zip folder by clicking the green botton labeled 
"code".
2. Use winZip or WinRar application to extract the contents of the zip file to your local folder.
3. Open Students Search.html by double clocking on it.
4. Select student's name on the list then click Search button.
5. You will be able to see students information below as you select each student's name from the list.
